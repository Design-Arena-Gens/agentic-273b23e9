# LumaDesk AI Concierge

Autonomous workplace receptionist that triages seat reservations, handles scheduling nuances, and automates hybrid work logistics in a single conversation.

## Quick start

```bash
npm install
npm run dev
```

## Key capabilities

- Conversational booking flow with contextual follow-ups
- Rule-based intent parsing for dates, time ranges, and amenities
- Real-time booking summary with confirmation guardrails
- Workspace showcase and operational telemetry panels

## Available scripts

- `npm run dev` – start local development server
- `npm run build` – create production build
- `npm start` – run production server
- `npm run lint` – lint source files
- `npm run format` – format with Prettier

Deploy on Vercel with:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-273b23e9
```
